# mini-http
This is the purest and simplest HTTP server, providing only file services and virtual directory functionality, but it is efficient and has a GUI. Lightweight and easy to use.

# Features
- File services
- Virtual directory functionality
- Efficient and lightweight
- GUI


### 安装 MinGW C 编译器
1. 下载 MinGW-w64 安装包： https://sourceforge.net/projects/mingw-w64/files/
2. 选择适合你系统的版本（建议选择 x86_64 版本）
3. 安装时确保选择 "posix" 线程模型和 "seh" 异常处理
4. 安装完成后，将 MinGW 的 bin 目录添加到系统 PATH 环境变量中
